# Security

If you think you have found a **security vulnerability**, please **do not** open a public GitHub issue with exploit details.

## How to report

Use **[GitHub private vulnerability reporting](https://github.com/matzekFloyd/GeoMania/security)** for this repository (open the **Security** tab, then *Report a vulnerability*), or reach the maintainers through another **private** channel if they have published one.

## What to include

- Steps to reproduce the issue  
- Affected area (desktop JAR, web app, CI, dependencies, etc.)  
- Impact and severity, if you can assess them  

You should receive a response through GitHub’s advisory workflow once the report is triaged.
